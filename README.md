# AoM Retold Build Orders

A static Age of Mythology: Retold build order site with pantheon selection, major god pages, build order cards, and an in-browser build order editor.

## Features

- Pantheon and major god selection homepage
- Major god build order library pages
- Detailed build order display tables
- Local build order editor using browser localStorage
- In-game inspired visuals and UI styling

## Running Locally

Use a local server instead of opening files directly.

```
bash
python3 -m http.server 8000

Then open:

http://localhost:8000


Important: run the site through a local server, not by double-clicking `index.html`, because the relative paths and browser storage behavior will be more consistent.

```
